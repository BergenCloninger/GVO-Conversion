The setup folder should contain: 
- Compiled Main.exe.
- teleapi.dll
- 68SERMC.DLL
- config.ini

Check For Devices: 
- Win + X -> Device Manager
- Expand Ports
- Check for some device (serial device, etc...)
- Set .ini to use that device port
